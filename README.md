Career Pathway Advisor App

## Overview
Career Pathway Advisor is an Android application designed to guide students in choosing the right career path after completing Class 12. 
It acts as a digital career counselor by providing structured guidance on courses, colleges, entrance exams, and skill development.

---

##  Problem Statement
Students often face confusion in selecting a career due to:
- Lack of proper guidance
- Scattered and unreliable information
- Missing important exam deadlines
- No centralized career planning platform

This application solves these issues by offering an all-in-one career guidance solution.

---

##  Features

### Career Assessment Test
- Categories: Aptitude, Reasoning, Verbal Ability
- Timer-based questions
- One question at a time
- Instant result analysis

---

### Stream-wise Career Guidance
Explore courses from:
- Science (MPC, BIPC, MBIPC)
- Commerce
- Arts
- Medicine
- Diploma & ITI
- Fine Arts

Includes:
- Course details & eligibility
- Career opportunities
- Colleges in Pondicherry
- Official website links

---

### Course Recommendation
- Suggests trending online courses
- Helps students improve skills
- External learning platform integration

---

###  Exam Calendar
- Tracks exams like JEE, NEET, NATA, CUET
- Displays important dates & deadlines
- Provides official registration links
- Reminder functionality

---

###  Resume Builder
- Create professional resumes
- Multiple templates
- Export as PDF

---

###  Interview Tips
- Common interview questions
- Resume & cover letter tips
- Communication & body language guidance

---

###  Admin Panel
- Add / Update / Delete courses
- Manage exam data
- Dynamic content control

## Technologies Used
- Java (Android Development)
- XML (UI Design)
- Android Studio
- Firebase Authentication
- Cloud Firestore

- No AI-based personalization
